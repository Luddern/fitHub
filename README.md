# Asp.net core & React

## 整體專案操作
1. > cd client
2. > npm run build
3. > copy /client/build/ ./wwwwroot
4. > cd ./
5. > dotnet run

go to web : https://localhost:7158


### 前端開發
1. > cd client
2. > npm run build
3. > npm start


### 後端開發
1. > ./
2. > dotnet run


### Run Docker
1. > docker build -t [image-name] .
2. > docker run -p [localhost:port]:80 --name [container-name]
